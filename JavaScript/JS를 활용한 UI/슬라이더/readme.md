## 🧑‍💻 새로 배운 것

- ~ 결합자 : 형제, 즉 첫 번째 요소를 뒤따르면서 같은 부모를 공유하는 두 번째 요소를 선택한다.
- JS에서 체크박스 체크여부 확인하기 : '선택자.checked', 체크여부에 따라 true or false 반환
