## 🧑‍💻 새로 배운 것

- padStart() : 현재 문자열의 시작을 다른 문자열로 채워, 주어진 길이를 만족하는 새로운 문자열을 반환한다.
  ```javascript
  const str = "5";
  console.log(str.padStart(2, "0"));
  // output: "05"
  ```
